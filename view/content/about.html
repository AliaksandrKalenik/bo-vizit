О номерах:

База отдыха «Визит» находится в 300 метрах от моря на территории, огороженной со всех сторон. На территории имеются площадки для отдыха, детские площадки, площадка-гриль, беседки для отдыха, душевые кабины, искусственный водоем, автостоянка. Корпус базы отдыха представляет собой двухэтажное здание, в котором расположены разнообразнейшие номера:

1-й этаж:


Комфорт двухместный:
Туалетная комната (круглосуточно хол./гор. вода), спутниковое ТВ, холодильник, мягкая удобная мебель, лоджия для отдыха.


Трехместный номер:
Мягкая удобная мебель, комнаты укомплектованы шкафами для одежды, столами, стульями. Для самостоятельного приготовления пищи – кухня, укомплектованная холодильниками, посудой, шкафами для хранения продуктов, мойками. Умывальник, душ, туалет находятся в отдельных строениях.



Четырехместный номер:
Мягкая удобная мебель, столы, стулья, спутниковое ТВ. Для самостоятельного приготовления пищи – кухня, укомплектованная холодильниками, посудой, шкафами для хранения продуктов, мойками. Умывальник, душ, туалет находятся в отдельных строениях.

2-й этаж:


двухместный и трехместный номера с лоджией:
Мягкая удобная мебель, комнаты, столами, стульями. Для самостоятельного приготовления пищи – кухня на этаже, укомплектованная холодильниками, посудой, шкафами для хранения продуктов, мойками. Умывальник, душ, туалет находятся в отдельных строениях.



Номер Люкс:
Туалетная комната (круглосуточно хол./гор. вода), спутниковое ТВ, холодильник, двуспальная кровать, диван, барная стойка, лоджия для отдыха, кондиционер.



Люкс двухэтажный пятиместный
Первый этаж разделен на две функциональные зоны - кухню и зону для отдыха. Кухня оборудована кухонной мебелью и посудой, холодильником, газовой плитой, мойкой для посуды. Зона для отдыха включает в себя диван - кровать, шкаф - купе, телевизор, кондиционер, ванную комнату. На втором этаже расположена мебель для отдыха:односпальные кровати и тумбочки.

Люкс двухэтажный четырехместный
Первый этаж разделен на две функциональные зоны - кухню и зону для отдыха. Кухня оборудована кухонной мебелью и посудой, холодильником, газовой плитой, мойкой для посуды. Зона для отдыха включает в себя диван - кровать, шкаф - купе, телевизор, кондиционер, ванную комнату. На втором этаже расположена мебель для отдыха:односпальные кровати и тумбочки.
